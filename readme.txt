== Description ==

This is the wordpress plugin for the [aidah.ai](https://aidah.ai) live chat widget.It's meant to make it easier to integrate aidah's real time live chat in WordPress sites. Connect with your websites visitors in a new, personal and exciting way.

Aidah Live Chat is a free and real-time chat for your website. With Aidah you can connect your Facebook messenger, slack, twitter(coming soon), to your dashboard and reply from one source. This is the official free Live Chat plugin for WordPress if you want to generate more leads, automate conversation and enhance your customer relationship within the same tool.

The free plugin includes live chat, Real-time notifications, unlimited chat, website triggers, Mobile apps, Availability schedule. All these features work immediately with no developer help needed.

Aidah also provides a robust AI chatbot tool that help you automate your Frequently Asked Questions. In addition we offer human chat agent that can help respond to your users 24/7 at 50cent/hour. 


== Installation ==

1. Install and activate the plugin.
2. Open the settings page of the plugin,you might see it in your sidebar after a quick refresh.
3. Enter your aidah live chat id,
4. Activate the plugin by selecting  yes  in the next option  and save click the save changes button.
